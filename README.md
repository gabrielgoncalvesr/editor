## Rich-Text Editor
A simple project of rich-text editor, using simple tools as html, css and js.

Main features:
- Header Text
- Font Type
- Size Font
- Color Font
- Backgroung Font
- Clear Content
- Print
- Undo
- Redo
- Delete Formatting
- Bold Text
- Underline Text
- Justify Left, Rigth and Center
- Numbered and Dotted List
- Quote Text
- Delete and Add Text Identation
- Hyperlink Text
- Cut
- Copy
